## 🕹️ Beyond the Code

When I'm not in class or working on Capstone, I'm building things on my own time — because the best way to actually learn is to just start making stuff.

**🤖 Generative AI & Bots**
I work with AI APIs from providers like OpenAI and Anthropic to build custom bots and integrations. I'm also going deeper — teaching myself machine learning fundamentals and running models **locally** using tools like [Ollama](https://ollama.com), so I'm not always dependent on a cloud API. The goal is to understand what's happening under the hood, not just call an endpoint.

**⚙️ C++ (Self-Taught)**
Picking up C++ on my own time to get closer to the metal. Coming from higher-level languages, it's humbling — but understanding memory management and lower-level systems makes everything else click better.

**🎮 Roguelikes & Retro Hardware**
I'm deeply into the permadeath loop of games like *Dead Cells* — every run is a lesson. I also take apart old gaming consoles just to see what makes them tick. It's the same instinct: figure out how things actually work, not just how to use them.
