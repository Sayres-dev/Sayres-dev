## 🕹️ Beyond the Code

When I'm not in class or working on Capstone, I'm building things on my own time — because the best way to actually learn is to just start making stuff.

**🎮 Game Development (C++ & Unity)**
I'm building an original game using Unity and C++ as a hands-on way to teach myself the language. Nothing forces you to understand memory management and systems thinking faster than trying to make something fun. The game lives under Lxquid Digital and will be linked here when it's far enough along to show off.

**🤖 Generative AI & Bots**
I work with AI APIs from providers like OpenAI and Anthropic to build custom bots and integrations. I'm also going deeper — teaching myself machine learning fundamentals and running models **locally** using tools like [Ollama](https://ollama.com), so I'm not always dependent on a cloud API. The goal is to understand what's happening under the hood, not just call an endpoint.

**⚙️ C++ (Self-Taught)**
Between the game project and low-level experimentation, C++ is my current white whale. Coming from higher-level languages, it's humbling — but understanding pointers and memory makes everything else click better.

**🔧 Retro Hardware & Roguelikes**
I take apart old gaming consoles just to see what makes them tick. Same instinct as everything else: figure out how things actually work. On the gaming side, the permadeath loop of *Dead Cells* hits different — every run teaches you something.

```
🎮 Game Dev (Unity + C++)   🤖 AI APIs & Local Models   ⚙️ C++ Self-Study   🔧 Retro Teardowns
```
